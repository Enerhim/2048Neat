# 2048 NEAT

This project implements the NEAT (NeuroEvolution of Augmenting Topologies) evolution algorithm to play 2048 recreated in PyGame using Python

Orginal Paper: [PDF](https://nn.cs.utexas.edu/downloads/papers/stanley.cec02.pdf)

NEAT-Python Documentation: [readthedocs.io](https://neat-python.readthedocs.io/en/latest/)
